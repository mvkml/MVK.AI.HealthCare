# HC.AI.MAPI.Services

## Layer
Service Layer (SL) — the API's own orchestration role, sitting directly under the Controller and
handing off to the Agent Layer. Extracted from a `Services/` folder inside `HC.AI.MAPI` to match
the rest of the solution's one-class-library-per-layer convention.
